# 반도체 웨이퍼 맵 결함 자동 분류 시스템
> **반도체 웨이퍼 제조 공정의 미세 결함 검출을 위한 딥러닝 모델 비교 연구**

반도체 미세 공정이 고도화됨에 따라 사람의 육안으로 식별하기 어려운 미세 결함을 자동으로 탐지하고 분류하기 위해 개발된 프로젝트입니다. 대용량 웨이퍼 맵 데이터를 활용하여 최적의 전처리 파이프라인을 구축하고, 3가지 딥러닝 아키텍처(CNN, ResNet, EfficientNet)의 성능을 비교 분석했습니다.

---

## 1. 프로젝트 개요 (Overview)

* **배경:** 반도체 웨이퍼 제조 공정 내 미세 결함은 작업자의 육안 검사 시 많은 시간과 비용이 소모되며 오분류 가능성이 존재합니다.
* **목적:** 딥러닝 기반 자동 분류를 통해 검사 공정의 효율성을 높이고 수율을 극대화합니다.
* **핵심 과제:** 1. 노이즈 및 데이터 불균형(Class Imbalance) 해소를 위한 5단계 전처리 실험
  2. 동일 조건 하에 3가지 주요 모델(CNN, ResNet, EfficientNet) 간 성능 비교 및 최적의 모델 도출

---

## 2. 데이터셋 정보 (Dataset)

* **데이터 소스:** Kaggle WM-811K 웨이퍼 맵 데이터셋 (원본 약 81만 장)
* **데이터 형태:** $64 \times 64$ 크기의 2차원 원형 웨이퍼 맵 배열 (값: `0`, `1`, `2`)
* **클래스 구성 (총 10개 클래스):**
  * `None`: 정상 데이터
  * `Defect (9개)`: Scratch, Center, Donut, Edge-Ring, Loc, Random 등 결함 유형 클래스

---

## 3. 데이터 전처리 파이프라인 실험 (P2 ~ P5)

미세 노이즈 제거 및 불균형 데이터셋 학습 최적화를 위해 단계별 전처리를 수행하고 성능을 평가했습니다.

1. **데이터 정제:** 2차원 데이터를 1차원으로 Flatten 한 뒤, 라벨이 누락된 유효하지 않은 데이터 제거
2. **[P2] Resize & 정규화:** 제각각이던 이미지 크기를 $64 \times 64$로 통일하고 Min-Max 정규화 진행 (**★ 최종 가장 높은 성능 기록**)
3. **[P3] 중앙값 필터 (Median Filter):** 센서 오차나 미세먼지로 인한 독립적 노이즈를 제거하고 결함 패턴 형태 보존
4. **[P4] 데이터 증강 (Data Augmentation):** 17만 장의 데이터 중 절대적으로 부족한 소수 불량 데이터를 인위적으로 증강하여 모델 학습 유도
5. **[P5] 클래스 가중치 (Class Weight):** 불량 데이터에 높은 가중치를 부과하여 실제 불량 패턴의 검출력(Recall) 극대화

> **실험 결과:** 전처리 기법 비교 분석 결과, **P2 (Resize & 정규화)** 단계를 거친 데이터셋이 가장 우수한 기초 성능을 보여 최종 모델 비교 분석용 데이터로 채택되었습니다.

## 4. 프로젝트 구조 (Project Structure)

```text
├── data/              # WM-811K 원본 및 전처리된 데이터셋 (약 17만 장 분할 완료)
├── utils/             # 데이터 정제, 64x64 리사이즈, 필터 처리, 데이터 증강 등 전처리 모듈
├── models/            # 비교 분석에 사용된 3가지 아키텍처 (CNN, ResNet, EfficientNet) 정의
├── train/             # 동일 파라미터 기반 모델 학습 (50 Epochs) 및 검증 스크립트
├── logs/              # 모델별 학습 Loss 및 Accuracy 추적 로그
└── results/           # 최종 학습 가중치 및 모델별 성능 평가지표 (F1-Score 등)
